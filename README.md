# TypeScript tailored standard ESLint config for compono

Install the thing in your project

```
npm add -D eslint-config-compono-ts
```

Add `.eslintrc` to your project.

```json
{
  "extends": "compono-ts",
  "rules": {
    // project specific rules go here
  }
}
```
